## Technologies Used
- **React**

## Setup
Clone the repository to your local machine.

### Client side

#### Install dependencies

```bash
cd client
npm install
```

#### Runnig the code

```bash
cd client
npm start
```

And the create-react-app react-scripts package will start the app on your browser.
